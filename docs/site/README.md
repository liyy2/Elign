# ELIGN Interactive Concept Page

This folder contains an animated webpage that explains ELIGN and FED-GRPO.

## Local preview

From the repository root:

```bash
python3 -m http.server 8000
```

Then open:

- `http://localhost:8000/docs/site/`

## Files

- `index.html`: page structure and content
- `styles.css`: visual system and animations
- `main.js`: interactions, reveal motion, and hero particle canvas
